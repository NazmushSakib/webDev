CSS Overview Learning.

Overflow : scroll -> always scroll.
Overflow : auto -> scroll will be available if needed.
Overflow : clip -> it will won't allow to scroll.
Overflow-x : hidden -> it will hide the horizontal scroll-bar.
Overflow-y : hidden -> it will hide the vertical scroll-bar.
white-space: nowrap; -> all text will be in a single line.
There are some difference between Overflow & Text-Overflow.
